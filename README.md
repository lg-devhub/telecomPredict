# 📊 Predição de Desistência de Clientes (Churn Prediction)

Projeto de Machine Learning desenvolvido em Python para prever a probabilidade de um cliente cancelar um serviço (churn), utilizando Random Forest e uma interface interativa com Streamlit.

---

# 🚀 Tecnologias Utilizadas

- Python 3.14
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib
- Venv (Virtual Environment)
